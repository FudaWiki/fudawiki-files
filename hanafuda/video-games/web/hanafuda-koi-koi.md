_Hanafuda Koi-Koi!_ (<span lang='ja'>花札 こいこい！</span>) is a web game published in 2016 by Yahoo.
It was developed by Aeria Games.

It features the game [Koi-Koi](/en/hanafuda/games/koi-koi) with both single player and online multiplayer modes. The game is only in Japanese.

_Coded-Dragon_ was the personal website of Austin Allman, which hosted an English-language [Koi-Koi](/en/hanafuda/games/koi-koi) web game published in 2020. It was developed by Adam "CodaHighland" Higerd.

The game was previously found at http://coded-dragon.com/hanafuda/lobby/, but the site has gone offline and the game is no longer available. No archived copies are known.

_Hanafuda Aikoi_ (<span lang='ja'>花札あいこい</span>) is an i-mode mobile phone game published in 2001.
It was developed by H.Sasayama of B.B.

## Features
- Screen size: 120x120 pixels
- Card size: 12x18 pixels

_Nihon no Asobi Series: Hanafuda_ (<span lang='ja'>日本の遊びシリーズ 花札</span>, "Games of Japan Series: Hanafuda") is a video game for Windows published in 2001, developed by FEEL-GOOD. The latest update was version 10.0 in 2013.

The game features the hanafuda games [Koi-Koi](/en/hanafuda/games/koi-koi), [Hana-Awase](/en/hanafuda/games/hana-awase) (called Bakappana in-game), [Hachi-Hachi](/en/hanafuda/games/hachi-hachi) and [Mushi](/en/hanafuda/games/mushi).

> For Kingo, Oicho-Kabu and Sashikomi, see [Nihon no Asobi Series: Kabufuda](https://fudawiki.org/en/kabufuda/video-games/win/nihon-no-asobi-series-kabufuda).
For Ogura Hyakunin Isshu and Iroha Karuta, see [Nihon no Asobi Series](/en/karuta/video-games/win/nihon-no-asobi-series).
{.is-info}

While the developer's website is no longer available, their software was hosted on Vector, where both paid and shareware versions are still available for download. In the free shareware version of Hanafuda, each game is limited to 6 rounds instead of 12.

## Installing and running the game
Despite its age, the game runs fine on Windows 11. However, as it still uses Shift-JIS encoding instead of UTF-8, the computer must have a Japanese locale or use a locale emulator to work correctly, and to display Japanese text. The installer may also run into problems when not executed in the correct locale.

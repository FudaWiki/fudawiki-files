_Mobairu Oicho-Kabu_ (<span lang='ja'>モバイルおいちょかぶ</span>, lit. "mobile Oicho-Kabu") is an Android/iOS game for kabufuda games, developed by Moco-Game.

It is an offshoot of the hanafuda video game [*Hanafuda gēmu dai shūgō*](/en/hanafuda/video-games/android-ios/hanafuda-gemu-dai-shugo) by the same developer.

## Available games
The following kabufuda games can be played in this app:

* [Oicho-Kabu](/en/kabufuda/games/oicho-kabu)
* [Kachi-Kachi](/en/kabufuda/games/kachi-kachi)

Both games include a game mode for beginners, in which it is easier to win.
